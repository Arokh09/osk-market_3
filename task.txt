Первый этап

Создать MySQL базу данных с таблицами Users, Skills, City;
Поля таблицы Users:
id;
name;
city_id;
Поля таблиц Skills и City:
id;
name;
Можно создать другие вспомогательные таблицы, при необходимости;
Заполнить таблицы несколькими записями;
Один пользователь может не иметь навыков или иметь их несколько. Место жительства может быть только одно;
Используя PDO для подключения к БД, отобразить на странице список всех пользователей в таком виде или при желании оформить таблицей: 
<имя>. Место жительства: <город>. [Навыки (<навык>[,<навык>])];
Сделать кнопку для добавления нового пользователя со случайным именем (именно именем а не набором символов), местожительством и набором навыков;
Весь код выложить на GitHub.

Второй этап

При клике на кнопку добавления пользователя, отправлять POST запрос через ajax c колбеком success. Колбек должен обновлять страницу. При возможности сделать обновление данных без перезагрузки страницы.

Третий этап

Сделать все тоже, что в первом и втором этапе, но с использованием фреймворка Yii2. Для доступа к данным использовать ActiveRecord. Данные вывести в таблицу DataTables (https://datatables.net/). CRUD генератор не использовать.При добавлении нового пользователя, желательно обновлять данные в таблице без перезагрузки страницы. На каждой строчке добавить кнопку для удаления записи с окном подтверждения (можно через обычный js confirm). После удаления данных, желательно обновить данные без перезагрузки страницы.
